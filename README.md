# My Online CV
 
* [Website](https://cvnibs-38b48.firebaseapp.com/)

## Getting Started

1. Clone git repo
2. Run:

```
npm install

and

npm start
```

### Prerequisites

Node

Npm/yarn

## Deployment

```
npm run build

or

yarn build
```

## Built With

* [React](https://reactjs.org/) - View library
* [Material-UI](http://www.material-ui.com/) - Some UI components
